Weight Statistics Analyzer
A Java program that calculates weight statistics and categorizes users based on health thresholds.
Key Features:
• Calculates Average, Highest, and Lowest weights.
• Categorizes members (Underweight/Normal/Overweight) and assigns grades (A-D).
• Uses Arrays, For-loops, and Conditional logic.
